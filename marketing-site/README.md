# Marketing Site

This folder contains the static marketing landing page for Pretium Investments.

Deploy to Render as a Static Site with the following settings:

- Repository: https://github.com/Rxahm/PretiumInvestment2.git
- Branch: main
- Root Directory: marketing-site
- Build Command: (leave blank — static files only)
- Publish Directory: .

Links in the navigation point to the deployed backend at https://pretiuminvestment2.onrender.com/.
